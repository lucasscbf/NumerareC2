# Numerare Assessoria Contábil

Projeto React + Vite + Tailwind + Framer Motion pronto para rodar localmente e publicar no GitHub Pages.

## Rodar localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy GitHub Pages

O projeto está configurado para o repositório `NumerareC`:

```bash
npm run build
npm run deploy
```

Link esperado:

```txt
https://lucasscbf.github.io/NumerareC/
```
